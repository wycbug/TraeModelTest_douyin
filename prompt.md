现在你的任务是利用 Cloudflare Workers 的 full-stack 能力配合抖音单视频解析 API 实现一个项目，前端允许用户输入需要解析的视频地址，点击解析按钮之后调用 API 获取解析数据，然后以表格形式展示出来，并允许用户导出为 excel 或者 csv 文件。

## 具体实现要求：

### 前端功能：

1. **视频链接输入**：
   - 提供输入框让用户粘贴抖音视频分享链接
   - 支持多种抖音链接格式识别
   - 输入验证和格式提示
2. **解析功能**：
   - 解析按钮触发 API 调用
   - 显示解析进度和加载状态
   - 错误处理和用户友好的错误提示
3. **数据展示**：
   - 以表格形式展示解析结果
   - 包含字段：作者、作者 ID、视频标题、封面、无水印链接、音乐链接、作者头像等
   - 视频封面和作者头像的缩略图预览
   - 无水印链接和音乐链接的点击复制功能
4. **导出功能**：
   - 支持导出为 Excel 文件
   - 支持导出为 CSV 文件
   - 导出内容包含所有解析到的数据
   - 可选择导出字段

### 后端逻辑：

1. 在`./worker/index.ts`中实现 API 代理
2. 接收前端传来的视频 URL 参数
3. 调用抖音解析 API 获取视频信息
4. 对返回数据进行格式化和处理
5. 实现错误处理和重试机制
6. 添加请求频率限制

### API 集成：

- 根据 apidoc.md 文档中的 API 规范实现
- 支持 GET 请求，参数包括 url（必填）
- 处理 API 返回的 JSON 数据，提取视频详细信息
- 处理可能的解析失败情况

### 用户体验优化：

- 链接格式自动识别和清理
- 解析结果实时预览
